# 치킨게임

https://sadie100.github.io/chicken-game/

치킨게임은 10분 내로 엔딩을 볼 수 있는 간단한 슈팅 웹 게임입니다.
꼬꼬가 되어 적들을 물리치고 아기 병아리를 되찾아 보아요

조작 방식 : 화살표 혹은 wasd로 조작, 스페이스바로 슈팅
